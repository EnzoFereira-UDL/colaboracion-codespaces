Bienvenido al Repositorio de Enzo
A continuación te dare las instrucciones a seguir para completar este código en conjunto

1- Crea una rama con tu nombre (sugerencia "feature/tu-nombre")
2. Crea un programa llamado saludo.py
3. En el programa haz que pida el nombre del usuario e imprima un saludo de bienvenida
5. Añade el repositorio original como upstream ("git add remote upstream https://github.com/EnzoFereira-UDL/colaboracion-codespaces")
4. Una vez terminado de programar, realizar un git add, luego un git commit y usar push
6. Realizar un pull request y esperar a respuesta
7. Cuando se realize el merge haz fetch a mi repositorio para actualizarlo y subelo a tu repositorio
